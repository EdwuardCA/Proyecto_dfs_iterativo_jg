# DFS Iterativo con Pila Propia

## Objetivo
Implementar DFS sin recursión usando una pila (Stack) implementada manualmente.

## Requisitos implementados
- Stack: push, pop, peek, is_empty, size
- Graph no dirigido con lista de adyacencia
- dfs_iterative(start)
- connected_components()
- has_cycle_undirected()

## Cómo ejecutar
```bash
python proyecto_dfs.py
